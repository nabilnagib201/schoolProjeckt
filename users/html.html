هو الهيكل الأساسي لأي صفحة ويب
ما هو HTML بالضبط؟
HTML = HyperText Markup Language
لغة الترميز لإنشاء هيكل الصفحات
و CSS إيه؟
CSS = Cascading Style Sheets
للتحكم في الشكل والتصميم
بدون CSS الصفحة تظهر هكذا:
نعم يجب تحسين الشكل!
عناصر HTML الأساسية:
<h1> إلى <h6> عناوين
<p> فقرات
<div> حاويات
<h1>عنوان رئيسي</h1>
ينشئ عنوان كبير ومهم
و <p> ؟
<p>هذا نص فقرة عادية</p>
للنصوص العادية والمحتوى
<div>محتوى</div>
الحاوية الأم لتجميع العناصر
هيكل HTML الأساسي:
<!DOCTYPE html>
<html>
  <head>
    <title>عنوان الصفحة</title>
  </head>
  <body>
    <h1>مرحباً</h1>
    <p>المحتوى</p>
  </body>
</html>
